Personal Portfolio Website

A modern and responsive full-stack personal portfolio website built using HTML, CSS, and JavaScript. This project is designed to showcase my skills, projects, education, certifications, and experience in an interactive and visually appealing manner.

📌 Project Overview

This portfolio website serves as a digital resume and professional profile. It provides visitors with information about my background, technical skills, projects, and contact details through a clean and modern user interface.

The project follows a modular folder structure and is designed to be scalable for future backend and database integration using technologies such as Node.js, Express.js, and MongoDB.
